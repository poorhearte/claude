---
description: 직전 어시스턴트 답변을 마크다운 원본 그대로 코드 펜스에 다시 출력
---

직전 어시스턴트 메시지(이 명령 직전에 네가 출력한 답변)를 **수정 없이** 마크다운 코드 펜스 안에 그대로 다시 출력해라.

규칙:
- 코드 펜스 외에는 어떠한 추가 텍스트(인사·요약·설명)도 붙이지 말 것
- 마크다운 문법이 렌더링되지 않은 원본 형태로 보여야 함
- 직전 답변 본문에 이미 ``` 코드 펜스가 들어있다면, 바깥쪽 펜스는 백틱 4개 이상(````)으로 감싸 중첩이 깨지지 않게 처리
- 직전 답변이 없거나 식별할 수 없으면 "직전 답변을 찾을 수 없습니다." 한 줄만 출력
